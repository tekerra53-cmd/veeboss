# VeeBoss Stitches Portfolio Website

Vite + React (TypeScript) single-page portfolio website with Tailwind CSS and some UI enhancements (e.g., Framer Motion).

## Prerequisites
- Node.js (LTS recommended)
- npm

## Install
```bash
npm install
```

## Start (development)
```bash
npm run dev
```
Then open the URL Vite prints in the terminal (e.g. `http://localhost:5173`).

> Note: If a port is already in use, Vite will automatically try another one.

## Build (production)
```bash
npm run build
```

## Preview production build
```bash
npm run preview
```

## Project structure
- `index.html` – app entry HTML
- `src/main.tsx` – React mount + providers
- `src/App.tsx` – main app component / routing
- `src/index.css` – global styles (Tailwind)
- `vite.config.ts` – Vite configuration
- `public/` – static assets

## Useful scripts
- `npm run dev` – start dev server
- `npm run build` – build for production
- `npm run preview` – preview built site
